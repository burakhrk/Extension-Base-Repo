# Chrome Extension Supabase Vite Starter

This is an opinionated starter meant to be copied into a real extension repo.

It includes:

- app constants
- Supabase client
- Chrome-compatible auth wrapper
- app-scoped social client
- analytics helper
- example bootstrap flow
- starter folders for `public/icons`, `scripts`, and `store-assets`

Adjust before using:

- `APP_ID`
- env values
- product-specific event names or screens
- actual UI integration
